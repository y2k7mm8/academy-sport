# Документация компонентов

## Обзор компонентов проекта ЦПМСК КР

### Layout компоненты

#### `<Layout />`
Главный layout-wrapper для всех страниц. Включает Header, Footer и область для контента.

**Использование:**
```tsx
<Layout>
  <YourPage />
</Layout>
```

#### `<Header />`
Навигационная панель с логотипом, меню и мобильной навигацией.
- Sticky позиционирование
- Адаптивное меню
- Индикатор активной страницы
- Национальная цветовая полоса сверху

#### `<Footer />`
Подвал сайта с контактной информацией, ссылками и партнерами.
- 4-колоночная сетка на десктопе
- Адаптивная верстка
- Использует `siteConfig` для данных

#### `<ScrollToTop />`
Автоматически прокручивает страницу вверх при переходе между роутами.

### UI компоненты

#### `<Button />`
Универсальная кнопка с поддержкой различных вариантов и размеров.

**Props:**
- `variant`: "primary" | "secondary" | "outline" | "ghost"
- `size`: "sm" | "md" | "lg"
- `href`: опциональная ссылка (становится Link компонентом)
- `icon`: иконка
- `iconPosition`: "left" | "right"

**Пример:**
```tsx
<Button variant="primary" size="md" href="/about" icon={<ArrowRight />}>
  Узнать больше
</Button>
```

#### `<Card />`
Контейнер для контента с различными вариантами оформления.

**Props:**
- `variant`: "default" | "bordered" | "elevated"
- `hover`: boolean - включить hover-эффекты

**Пример:**
```tsx
<Card variant="bordered" hover>
  <h3>Заголовок</h3>
  <p>Содержимое карточки</p>
</Card>
```

#### `<NationalColorBar />`
Декоративная полоса национальных цветов Кыргызстана.

**Props:**
- `height`: строка с Tailwind-классом высоты (по умолчанию "h-1.5")

**Пример:**
```tsx
<NationalColorBar height="h-2" />
```

#### `<Loader />`
Индикатор загрузки с национальными цветами.

**Props:**
- `size`: "sm" | "md" | "lg"

### Секционные компоненты

#### `<PageHero />`
Hero-секция для внутренних страниц с единообразным дизайном.

**Props:**
- `category`: категория страницы
- `categoryColor`: "red" | "yellow"
- `title`: заголовок (ReactNode)
- `subtitle`: подзаголовок
- `borderColor`: "red" | "yellow"
- `backgroundImage`: URL фонового изображения
- `minHeight`: строка с Tailwind-классом

**Пример:**
```tsx
<PageHero
  category="О Центре"
  categoryColor="red"
  title={<>ИСТОРИЯ <br/><span className="text-neutral-500">И МИССИЯ</span></>}
  subtitle="Главное учреждение Кыргызской Республики..."
  borderColor="yellow"
  backgroundImage="/path/to/image.jpg"
/>
```

#### `<SectionHeader />`
Заголовок секции с метками и описанием.

**Props:**
- `label`: метка секции
- `labelColor`: "red" | "yellow" | "white"
- `title`: заголовок (ReactNode)
- `subtitle`: подзаголовок
- `align`: "left" | "center"

**Пример:**
```tsx
<SectionHeader
  label="Направления"
  labelColor="red"
  title={<>НАШИ <span className="text-[#E31837]">ЦЕННОСТИ</span></>}
  subtitle="Фундаментальные принципы работы центра"
  align="left"
/>
```

#### `<StatCard />`
Карточка для отображения статистики.

**Props:**
- `value`: значение статистики
- `label`: описание
- `accentColor`: "red" | "yellow"

**Пример:**
```tsx
<StatCard 
  value="500+" 
  label="СПОРТСМЕНОВ В ПОДГОТОВКЕ"
  accentColor="red"
/>
```

### Константы и конфигурация

#### `siteConfig`
Глобальная конфигурация сайта (адреса, контакты, навигация).

**Импорт:**
```tsx
import { siteConfig } from '../constants/siteConfig';
```

Национальные цвета Кыргызстана.

```tsx
import { NATIONAL_COLORS } from '../constants/theme';

// NATIONAL_COLORS.yellow = '#FFCD00'
```

### Утилиты

#### `cn(...inputs)`
Объединение Tailwind-классов с поддержкой условной логики.

**Использование:**
```tsx
import { cn } from '../utils/cn';

<div className={cn(
  "base-class",
  isActive && "active-class",
  variant === "primary" && "primary-class"
)} />
```

#### `formatDate(dateString)`
Форматирование даты в русском формате.

```tsx
import { formatDate } from '../utils/formatDate';
formatDate("2026-03-16") // "16 марта 2026"
```

#### `formatDateShort(dateString)`
Краткое форматирование даты.

```tsx
formatDateShort("2026-03-16") // "16.03.2026"
```

## Стилевые принципы

### Цвета
- **Красный (#E31837)** - основной акцент, используется для важных элементов
- **Желтый (#FFCD00)** - вторичный акцент, используется для выделения
- **Черный/Темно-серый** - для темных секций и текста
- **Белый** - основной фон

### Типографика
- Все заголовки в верхнем регистре (`uppercase`)
- Жирное начертание для заголовков (`font-black`)
- Увеличенный межбуквенный интервал (`tracking-widest`, `tracking-tighter`)
- Мобильно-адаптивные размеры шрифтов

### Анимации
- Плавные переходы 300ms для hover-эффектов
- Grayscale-фильтры с раскрытием цвета при наведении
- Transform-анимации для интерактивных элементов

### Spacing
- Секции: `py-24 md:py-32`
- Контейнеры: `max-w-[1440px] mx-auto px-4 sm:px-6 lg:px-12`

## Лучшие практики

1. **Используйте переиспользуемые компоненты** из `/src/app/components/`
2. **Импортируйте константы** из `siteConfig` вместо хардкода
3. **Применяйте утилиту cn()** для условных классов
4. **Следуйте адаптивному подходу** mobile-first
5. **Используйте национальные цвета** для акцентов
6. **Применяйте консистентный spacing** из констант
7. **TypeScript типы** обязательны для всех пропсов
