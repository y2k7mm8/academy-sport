## О проекте

Современный, профессиональный веб-сайт, выполненный в строгом официальном стиле с использованием **национальных цветов Кыргызской Республики** (красный #E31837 и желтый #FFCD00). Дизайн вдохновлен профессиональными спортивными платформами, такими как olympic.org, с акцентом на:

- 🏛️ Архитектурную сетку и строгую геометрию
- 🎨 Контрастные блоки и массивную типографику  
- 🇰🇬 Национальную символику и идентичность
- 📱 Полную адаптивность для всех устройств
- ⚡ Высокую производительность и доступность

## Быстрый старт

### Требования

- Node.js 18+ 
- npm или pnpm

### Установка

```bash
# Клонирование репозитория
git clone <repository-url>
cd sports-center-kg

# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

### Сборка для продакшена

```bash
# Production build
npm run build

# Предпросмотр build
npm run preview
```

## Дизайн-система

### Цветовая палитра

```css
/* Национальные цвета */
--national-red: #E31837;    /* Основной акцент */
--national-yellow: #FFCD00; /* Вторичный акцент */

/* Нейтральные */
--white: #FFFFFF;
--black: #0A0A0A;
--neutral-50 to 950
```

### Типографика

- **Заголовки**: font-black, uppercase, tracking-tighter
- **Подзаголовки**: font-bold, uppercase, tracking-widest
- **Текст**: font-medium, оптимальная высота строки

### Ключевые принципы

1. **Минимализм** - чистый, неперегруженный дизайн
2. **Контраст** - четкое разделение через цвет и размер
3. **Геометрия** - архитектурная сетка
4. **Национальность** - красный и желтый как статусные акценты
5. **Адаптивность** - mobile-first подход
