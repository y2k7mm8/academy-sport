# Центр подготовки молодежных сборных команд КР

Официальный сайт для Центра подготовки молодежных сборных команд при Государственном агентстве физической культуры и спорта при Кабинете Министров Кыргызской Республики.

## Описание проекта

Современный, профессиональный веб-сайт, выполненный в строгом официальном стиле с использованием национальных цветов Кыргызской Республики (красный #E31837 и желтый #FFCD00). Дизайн вдохновлен профессиональными спортивными платформами, такими как olympic.org, с акцентом на архитектурную сетку, контрастные блоки и массивную типографику.

## Технологический стек

- **React 18** - библиотека для построения пользовательского интерфейса
- **TypeScript** - типизированный JavaScript
- **React Router 7** - клиентская маршрутизация
- **Tailwind CSS v4** - утилитарный CSS-фреймворк
- **Vite** - современный сборщик и dev-сервер
- **Lucide React** - иконки

## Основные страницы

1. **Главная (/)** - Hero-секция, статистика, обзор видов спорта, CTA
2. **О Центре (/about)** - Миссия, видение, ценности, история, структура организации
3. **Виды спорта (/sports)** - Детальная информация по каждому виду спорта, требования к кандидатам
4. **Достижения (/achievements)** - Медальный зачет, хронология побед, выдающиеся атлеты
5. **Новости (/news)** - Пресс-релизы, события, фильтрация по категориям
6. **Контакты (/contact)** - Контактная информация, форма обращения
7. **404 (/*)** - Страница для несуществующих маршрутов

## Дизайн-система

### Цветовая палитра

**Национальные цвета:**
- Красный: `#E31837` - основной акцентный цвет
- Желтый: `#FFCD00` - вторичный акцентный цвет

**Нейтральные цвета:**
- Белый: `#FFFFFF` - фон
- Черный: `#0A0A0A` - текст и темные секции
- Серые оттенки: от `#F5F5F5` до `#171717`

### Типографика

- **Заголовки**: Массивные, в верхнем регистре, жирные (font-black)
- **Подзаголовки**: Средний вес, увеличенный межбуквенный интервал (tracking-widest)
- **Текст**: Среднее начертание (font-medium), оптимальная высота строки

### Ключевые дизайн-принципы

1. **Строгость и профессионализм** - минималистичный, но мощный визуальный язык
2. **Контраст** - четкое разделение секций через цвет и размер
3. **Архитектурная сетка** - геометрически выверенная компоновка
4. **Национальная идентичность** - использование красного и желтого как статусных акцентов
5. **Адаптивность** - полная поддержка мобильных устройств

## Компоненты

### Переиспользуемые компоненты

- `<NationalColorBar />` - декоративная полоса национальных цветов
- `<PageHero />` - hero-секция для внутренних страниц
- `<SectionHeader />` - заголовки секций с консистентным стилем
- `<StatCard />` - карточки для отображения статистики
- `<Button />` - кнопки с различными вариантами
- `<Card />` - контейнеры для контента
- `<ScrollToTop />` - автоматическая прокрутка вверх при смене страницы

### Layout компоненты

- `<Header />` - навигация с sticky-позиционированием
- `<Footer />` - подвал с контактами и ссылками
- `<Layout />` - общий layout-wrapper для всех страниц

## Особенности реализации

### React Router

Проект использует React Router 7 в режиме Data API с `createBrowserRouter`. Все маршруты определены в `/src/app/routes.tsx`.

### Responsive Design

Все компоненты адаптированы для экранов от 320px (мобильные) до 1440px+ (десктоп). Используется мобильно-ориентированный подход (mobile-first).

### Изображения

- Используется компонент `<ImageWithFallback />` для безопасной загрузки изображений
- Все изображения оптимизированы через Unsplash CDN
- Применяется lazy loading для производительности

### Анимации

- Плавные переходы между состояниями (transition)
- Hover-эффекты на интерактивных элементах
- Grayscale-фильтры с цветовым раскрытием при наведении
- Smooth scroll для якорных ссылок

## Стилизация

### Tailwind CSS v4

Проект использует последнюю версию Tailwind CSS с кастомными CSS-переменными, определенными в `/src/styles/theme.css`.

### Кастомные стили

- Кастомные скроллбары с национальными цветами
- Smooth scroll behavior
- Утилитарный класс `.no-scrollbar` для скрытия полос прокрутки
